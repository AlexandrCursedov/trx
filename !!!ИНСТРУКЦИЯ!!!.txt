Для начала работы нужно установить NodeJS и VS Code
1. установи NodeJS - https://nodejs.org/en/download
2. установи VS Code - https://code.visualstudio.com/download

3. Перезагрузи пк чтобы изменения вошли в силу
4. Открой файл trxswiper.js через VS Code
5. Открой терминал выше
6. Установи необходимые библиотеки командой -> npm install tronweb tronweb-tool telegraf fs path bip39
7. Дождись установки библиотек
8. В файле trxswiper.js поменяй КОНФИГ под свои данные (в мануале более подробно написано как и что)
9. Запусти скрипт командой -> node trxswiper.js и не закрывай VS Code
10. Готово! теперь скрипт каждые 10 секунд проверяет баланс и выводит его на третий кошелек где будет чистая прибыль
